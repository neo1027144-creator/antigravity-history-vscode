# Contributing to Antigravity History

Thanks for your interest in contributing! This project is open source under the Apache 2.0 license and welcomes contributions of all kinds.

## Ways to Contribute

- 🐛 **Report Bugs** — [Open an issue](https://github.com/neo1027144-creator/antigravity-history-vscode/issues) with steps to reproduce
- 💡 **Suggest Features** — Share your ideas via issues
- 📝 **Improve Docs** — Fix typos, clarify instructions, add examples
- 💻 **Submit Code** — Bug fixes, new features, or performance improvements

## Development Setup

```bash
git clone https://github.com/neo1027144-creator/antigravity-history-vscode.git
cd antigravity-history-vscode
npm install
npm run build
```

To test locally:
1. Press `F5` in VS Code to launch the Extension Development Host
2. Or run `vsce package --no-yarn` to build a `.vsix` file

## Pull Request Guidelines

1. Fork the repo and create a feature branch
2. Keep changes focused — one PR per feature/fix
3. Follow the existing code style
4. Test your changes before submitting
5. Write a clear PR description

## Code of Conduct

Be respectful, inclusive, and constructive. We're all here to build something useful.

## Questions?

Feel free to open an issue or start a discussion. No question is too small.
